# 🌺 Bella House - E-commerce Store

Bella House is a modern e-commerce storefront built with Next.js, Tailwind CSS, GSAP, and Supabase. It combines responsive design, smooth scroll animations, and dynamic content to create an engaging shopping experience.

## ✨ Description

A polished online retail interface for Bella House, featuring animated page transitions, policy-driven content pages, and an integrated user experience built for fast performance and flexibility.

## 🚀 Key Features

- Smooth canvas scroll animation and motion-driven transitions
- Dynamic policy pages powered by route parameters
- Responsive product, shop, checkout, account, and contact pages
- Authentication and cart support with Supabase
- Tailwind CSS styling with modern responsive layouts
- Built on Next.js App Router for server-side rendering and fast performance

## 🧰 Tech Stack

- Next.js
- Tailwind CSS
- GSAP
- Supabase

## 📦 Installation

```bash
git clone https://github.com/arjunkotwal153/Bella-House.git
cd Bella-House
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.
